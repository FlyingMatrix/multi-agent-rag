# Multi-agent RAG

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

### 🎯 About this repo

**multi-agent-rag** is a modular, extensible framework for building advanced Retrieval-Augmented Generation (RAG) systems powered by multiple collaborating agents, which combines modern LLM orchestration tools with efficient vector search and local model serving to enable scalable, intelligent workflows. 

The goal of **multi-agent-rag** is to provide a practical and extensible foundation for experimenting with next-generation AI systems where multiple agents collaborate to retrieve, reason, and generate high-quality responses.

This repository leverages:

- **LlamaIndex** for structured data ingestion and retrieval pipelines
- **ChromaDB** as a lightweight, high-performance vector database
- **LangChain** and **LangGraph** for agent orchestration and workflow design
- **Ollama** for running local large language models
- **Typer** for building intuitive CLI interfaces
- **Rich** for enhanced terminal output and debugging

## 🚀 Key features

- Multi-agent architecture for complex reasoning and task decomposition
- Flexible RAG pipelines with pluggable components
- Local-first LLM support via Ollama
- Graph-based workflows for transparent agent interactions
- Developer-friendly CLI tools and rich terminal visualization
